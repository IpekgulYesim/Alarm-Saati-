Bu Arduino R3 kullanılarak yapılmış bir alarm saatidir.
Devresi için: DS1307 yerine DS3231'i bağlayın. Arduino'nuzda SCL/SDA pinleri ayrıca varsa SCL ve SDA pinlerini adlarıyla eşleşen yerlere bağlayın. Yoksa SDA'yı A4, SCL'yi A5'e bağlayın.
Çalışması için:
* Kodu kopyalayın.
* Devreyi tamamladıktan sonra Arduino'nuzu bilgisayara bağlayın.
* Kodu 'Upload' tuşuna basarak Arduino'nuza yükleyin.
